TROUBLESHOOTING QUICK FIXES
================================================================================

PROBLEM: OG image not showing in Kakao Talk
FIX:
  1. Verify image size exactly: 1200x630 or 800x800
  2. Ensure file < 5MB
  3. Test at: https://www.opengraph.xyz/
  4. Clear Kakao cache: App Settings > Cache > Clear

PROBLEM: Schema.org not recognized
FIX:
  1. Validate JSON at: https://jsonlint.com/
  2. Check script in <head>: type="application/ld+json"
  3. Test at: https://search.google.com/test/rich-results
  4. Check Search Console for error messages

PROBLEM: Meta tags not updating in search results
FIX:
  1. Takes 2-4 weeks for Google to re-crawl
  2. Request re-indexing in Google Search Console
  3. Request Naver crawl in Naver Webmaster
  4. Verify index: true in robots config
  5. Check no duplicate canonical URL issues

PROBLEM: CTR not improving after 2 weeks
FIX:
  1. Verify changes deployed correctly
  2. Check Naver/Google showing new title/description
  3. Test for display issues with OG debugger
  4. Consider A/B testing other variations
  5. Check for keyword relevance to current content
