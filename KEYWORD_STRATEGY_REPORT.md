## Executive Summary

Luna Alba targets a highly niche but lucrative Korean market segment: women seeking flexible night work with privacy protection. This report provides data-driven keyword strategy for landing page optimization, focusing on semantic relevance, search intent alignment, and topical authority.

**Key Findings:**
- Primary keywords have moderate competition but high commercial intent
- Significant opportunity in long-tail variations and intent-specific phrases
- Community-driven keywords unlock retention and brand differentiation

### Gap Analysis Methodology

#### Phase 1: Competitor Keyword Discovery
**Tools to use:**
- Ahrefs / SEMrush (keyword research)
- Keyword gaps tool
- Manual search analysis

**Process:**
```
1. Identify 10-15 primary keywords for each competitor
2. Map keyword rankings (position, search volume, difficulty)
3. Calculate keyword overlap percentage
4. Identify unique competitor keywords
5. Assess Luna Alba keyword coverage
```

#### Phase 2: Keyword Gap Identification

**Gap Category A: Keywords Competitors Rank For But Luna Alba Doesn't**
```
Process:
1. Extract competitor ranking keywords
2. Filter for commercial/transactional intent
3. Remove low-relevance keywords
4. Prioritize high-volume, high-intent keywords

Example findings (Hypothetical):
- 퀸알바: "최고 시급 밤알바", "급할 때 밤알바"
- 온니잡: "단기 밤알바", "스터디 그룹"
- 레이디알바: "밤알바 안전 검증", "신원보호 알바"

Action: Create content targeting these gaps
```

**Gap Category B: Keywords Luna Alba Ranks For But Competitors Don't**
```
Process:
1. Identify Luna Alba keyword strengths
2. Confirm competitor absence in rankings
3. Calculate search volume and opportunity score
4. Plan content expansion

Example findings (Hypothetical):
- Luna Alba unique: "AI 매칭 밤알바", "프리미엄 여성알바"
- Defensibility: High (brand/tech differentiator)
- Action: Strengthen content moat

Opportunity Score = Search Volume × (1 - Competitor Count/5)
```

**Gap Category C: Emerging Keywords (Low Volume But Growing)**
```
Process:
1. Identify seasonal/trending keywords
2. Check Google Trends volume trajectory
3. Assess competitor adoption rate
4. Plan early-mover advantage

Example findings (Hypothetical):
- 2025 Emerging: "챗봇 알바 매칭", "VR 면접 연습"
- Growth rate: +40% YoY
- Competitor adoption: 20%
- Action: Create content on AI matching differentiation
```

## 11. Korean SEO Best Practices

### Linguistic Considerations

**Hangul Keyword Considerations:**
```
1. Spacing & Compound Words
   Correct: "밤알바 여성" (separate)
   Alternative: "밤알바여성" (compound)
   Usage: Use spaces for better keyword recognition

2. Character Encoding
   Ensure: UTF-8 encoding for all content
   Meta charset: <meta charset="UTF-8">
   Test: Korean text renders correctly across browsers

3. Particle Variations
   "밤알바를" (direct object particle)
   "밤알바는" (topic particle)
   Usage: Include natural particle variations in content

4. Jamo Decomposition
   Noto Sans Korean font for proper display
   Mobile devices: Verify hangul rendering
```

### Korean SEO Technical Requirements

**Server & Language Tags:**
```html
<html lang="ko">  <!-- Critical for Korean content -->
<meta charset="UTF-8">  <!-- UTF-8 encoding required -->
<meta http-equiv="content-language" content="ko-KR">
```
